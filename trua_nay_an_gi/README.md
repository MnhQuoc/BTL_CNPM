## Công nghệ:

  - Framework: React 19
  - Build tool: Vite
  - Routing: React Router DOM
  - UI Components: Ant Design, React Bootstrap
  - HTTP Client: Axios
  - Icons: React Icons
  - Email Service: EmailJS
  - Styling: SASS


## Scripts

- Khởi chạy dự án ở chế độ development:
  ```bash
  npm run dev
  ```

- Build dự án cho production:
  ```bash
  npm run build
  ```

- Kiểm tra lỗi code:
  ```bash
  npm run lint
  ```

- Preview bản build:
  ```bash
  npm run preview
  ```
